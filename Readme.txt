Scientific Calculator Assignment

Project Overview
The Scientific Calculator Assignment project extends the basic calculator functionalities to include a set of scientific features. In addition to standard arithmetic operations, this calculator incorporates advanced functions catering to scientific and engineering needs.

Features
- Basic Operations: 
	Addition, subtraction, multiplication, and division.
- Scientific Functions:
  - Trigonometric operations (sin, cos, tan)
  - Logarithmic functions (log, ln)
  - Exponential functions (e^x)
  - Square root and nth root
  - Factorial and more

User-Friendly Interface:
	Clear display and intuitive buttons for both basic and scientific operations.
Error Handling:
	Gracefully handle invalid inputs to ensure a smooth user experience.

How to Use
1. Clone the repository: https://github.com/khatriYT
2. Open the project in your preferred programming environment.
3. Run the program and follow on-screen instructions.

Technologies Used
- Technologies Used
Frontend: Java [Java Swing NetBeans]
